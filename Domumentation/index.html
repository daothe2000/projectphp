How to edit

These PSD files are well arranged, so it is very easy to edit. The PSD files are created in Photoshop version CC-2017. You can edit the files even with minimum knowledge in photoshop. The layers are well packed in folders and labled with : syembels, so you can easly idetify master folder and subfolders.

Text can be edit by clicking the text using ‘type tool’. Image can be added by just placing the images over the place holder layer and just mask them using layer masking.

arranged layers

All layers are wellpacked in folders. The labeling make it easy to identify masterfolders and sub folders

effects adjustment

It is easy to adjust the layer effects/Styles by double clicking the layer and adjust or add effects form the Layer Style window.
